步骤：

  1. 注册Github账号，创建同名仓库(用户名.github.io)
  2. 安装配置git
  3. 生成密钥
  4. 下载node.js 安装hexo，实现通过 localhost：4000 访问博客
  5. 和github关联推代码到github，实现通过 用户名.github.io 访问博客
  6. 换个自己喜欢的主题发第一篇博客

问题1: 执行 hexo d 报错

问题2: 再次执行hexo d 报错

解决方案

再次执行hexo d 发现不会报错啦，网站已经部署好了，可以在浏览器输入你的GitHub名称.github.io访问博客了。

更换主题

  • 在hexo官网主题中心https://hexo.io/themes/下载主题，找到喜欢的主题
  • 进入github，github命令下载
  • 命令行cd到hexo的themes目录下，git clone 主题GitHub地址
  • 修改_config.yml文件的 theme: 主题名 （注意：后有空格）
